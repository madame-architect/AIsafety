# AIsafety

> **Status:** Draft, active work-in-progress repository for AI safety literature review artifacts.

This repository contains research outputs, methods logs, and evidence-atlas deliverables used during an ongoing literature review process.

## ⚠️ Important disclaimer

This work is **not finalized or published**. Materials may be incomplete, revised, or replaced.
Use all contents at your own risk.

## What is in this repository

- **Main evidence atlas package:** `assurance_evidence_atlas_v1.0.0/`
  - Methodology, codebook, changelog, protocol lock, schemas, and exported datasets.
- **Research reports:** `deep-research-report*.md`
- **Search/process artifacts:** `method/search-log.csv`
- **Template asset:** `assurance_evidence_atlas_template_v1.xlsx`
