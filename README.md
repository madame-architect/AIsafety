# AIsafety

> **Status:** Draft, active work-in-progress repository for AI safety literature review artifacts.

This repository contains research outputs, methods logs, and evidence-atlas deliverables used during an ongoing literature review process.

## ⚠️ Important disclaimer

This work is **not finalized or published**. Materials may be incomplete, revised, or replaced.
Use all contents at your own risk.

## What's in this Repository

- `ai-safety-Literature_Review.md` — the main narrative literature review draft.
- `ai-safety-literature_review.bib` — the bibliography database used to track and cite sources.
- `LICENSE` — licensing terms for materials in this repository.
- `README.md` — repository overview, status, and audience guidance.


## Intended audience

- Researchers reviewing AI safety assurance evidence.
- Collaborators contributing to curation and adjudication.
- Readers evaluating methodology and dataset provenance.
